# IPXER-API

This directory contains the source code for the `shaper-api` binary.

## Purpose

The `shaper-api` binary is an HTTP server that provides an API for managing iPXE configurations. It is responsible for:

- Resolving and transforming iPXE configurations.
- Serving iPXE scripts to clients.
- Exposing metrics for monitoring.
